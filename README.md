# APP - Mata Mosquito
Game interativo - Javascript

### Detalhes
- Use uma raquete para matar todos os mosquitos do bosque;
- HTML | CSS | Javascript

### Descrição
- Escolha o nível;
- Mate todos os mosquitos;
- Cada raquetada errada, você perde uma vida;
- Finalize todos os níveis.
